    This React component creates a Wikipedia-style biography page that closely resembles the layout and content shown in the images. It includes:

    1. A header with the Wikipedia logo and navigation icons
    2. The main title (Chitron Bhattacharya)
    3. An infobox with key details about the person
    4. Collapsible sections for Early Life, Education and Career, and Projects
    5. The content is primarily in Bengali, as shown in the images

    The component uses Tailwind CSS for styling and includes some basic interactivity (although the collapsible sections are not fully functional in this example).
